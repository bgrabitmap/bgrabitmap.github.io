Description

Layered image, each layer being a TBGRABitmap or rendered from an original. It can handle SVG format, gradients and blend modes.

Uses

  • BGRAGraphics
  • BGRAClasses
  • SysUtils
  • BGRABitmapTypes
  • BGRABitmap
  • BGRAMemDirectory
  • BGRATransform
  • fgl
  • BGRALayerOriginal
